# KNW-019 Weather Widget Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Show current weather (icon + temperature + city) on the Dashboard next to the greeting, using browser geolocation and OpenWeatherMap.

**Architecture:** Client-side geolocation feeds lat/lon to a server-side proxy route `/api/weather` (which holds the API key). Dashboard fetches weather after mount; any failure silently hides the widget.

**Tech Stack:** Next.js 16 App Router, TypeScript, Tailwind CSS, OpenWeatherMap Current Weather API

- [ ] **Step 2: Create the route file**

```ts
import { NextRequest, NextResponse } from "next/server"
import { auth } from "@/auth"

export const runtime = "nodejs"

export async function GET(request: NextRequest) {
  const session = await auth()
  if (!session?.user?.id) {
    return NextResponse.json({ error: "Unauthorized" }, { status: 401 })
  }

  const { searchParams } = request.nextUrl
  const lat = parseFloat(searchParams.get("lat") ?? "")
  const lon = parseFloat(searchParams.get("lon") ?? "")

  if (isNaN(lat) || isNaN(lon) || lat < -90 || lat > 90 || lon < -180 || lon > 180) {
    return NextResponse.json({ error: "Invalid coordinates" }, { status: 400 })
  }

  const apiKey = process.env.OPENWEATHERMAP_API_KEY
  if (!apiKey) {
    return NextResponse.json({ error: "Weather API not configured" }, { status: 503 })
  }

  try {
    const res = await fetch(
      `https://api.openweathermap.org/data/2.5/weather?lat=${lat}&lon=${lon}&appid=${apiKey}&units=metric&lang=zh_cn`,
      { next: { revalidate: 600 } } // cache 10 min
    )
    if (!res.ok) {
      return NextResponse.json({ error: "Weather fetch failed" }, { status: 502 })
    }
    const data = await res.json()
    return NextResponse.json({
      temp: Math.round(data.main.temp),
      description: data.weather[0]?.description ?? "",
      icon: data.weather[0]?.icon ?? "",
      city: data.name ?? "",
    })
  } catch {
    return NextResponse.json({ error: "Weather fetch failed" }, { status: 502 })
  }
}
```

- [ ] **Step 3: Add weather fetch useEffect**

```tsx
useEffect(() => {
  if (!navigator.geolocation) return
  navigator.geolocation.getCurrentPosition(
    (pos) => {
      const { latitude: lat, longitude: lon } = pos.coords
      fetch(`/api/weather?lat=${lat}&lon=${lon}`)
        .then((r) => (r.ok ? r.json() : null))
        .then((d) => { if (d) setWeather(d) })
        .catch(() => {})
    },
    () => {} // silently ignore permission denied
  )
}, [])
```

```tsx
function weatherEmoji(icon: string) {
  const code = icon.slice(0, 2)
  const map: Record<string, string> = {
    "01": "☀️", "02": "⛅", "03": "☁️", "04": "☁️",
    "09": "🌧️", "10": "🌧️", "11": "⛈️", "13": "🌨️", "50": "🌫️",
  }
  return map[code] ?? "🌡️"
}
```
